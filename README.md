## Установка

## Для синхронизации с БД необходимо создать Табличку в MYSQL и прописать данные в .env
### Node сервер (корень проекта)
```
npm install
```

## Сборка и запуск проекта
### Запуск сервера (корень проекта)
```
nodemon server.js
```

## Debug
### Включить
```
set DEBUG=express:* & node server.js
```

### Выключить
```
set DEBUG= server.js
```
